Design a modern cybersecurity-themed website UI for a **Password Strength Checker Web Application**.

The design should follow a **dark theme** with:

* Primary color: Black (#0a0a0a)
* Secondary color: Dark grey (#1f1f1f)
* Accent color: Neon blue (#00c8ff)

The style should feel similar to modern cybersecurity dashboards and developer tools.

---

LAYOUT STRUCTURE

Create the following sections in order:

1. HERO SECTION

* Large headline: "Check Your Password Strength Instantly"
* Subtext explaining why strong passwords matter
* Password input box with "Check Strength" button
* Animated strength meter (weak → strong)
* Subtle glow effects

2. HOW IT WORKS

* 3 step cards:

  1. Enter your password
  2. AI analyzes the strength
  3. Get security recommendations
* Include icons for each step.

3. PASSWORD SECURITY GUIDE
   Create a section explaining:

* What makes a strong password
* Common password mistakes
* Best practices for password security

Design this as cards or expandable accordion blocks.

4. VIDEO TUTORIAL SECTION
   Add a section titled:

"How to Create Strong Passwords"

Include:

* Placeholder YouTube video frame
* Description text
* Callout tips

5. FEATURES SECTION
   Highlight key features:

* Real-time strength analysis
* Entropy calculation
* Crack-time estimation
* Security suggestions

Use glowing cards with hover effects.

6. FOOTER
   Include:

* GitHub link
* Documentation
* Contact
* Copyright notice

---

Design with animations in mind:

* Scroll reveal animations
* Fade-in cards
* Hover glow effects
* Animated progress bar for password strength
